# K-8 Mathematics Title Development Workflow

Standard operating procedure for developing new K-8 mathematics titles
from manuscript through publication.

## Phase 1: Content Development (Months 1-8)

### 1.1 Planning & Scoping

**Duration:** 4-6 weeks

1. Define scope and grade level
2. Identify target standards (CCSS, state-specific)
3. Assemble author team
4. Create detailed table of contents
5. Establish page budget and format

**Deliverables:**
- [ ] Signed author contracts
- [ ] Detailed TOC with page allocations
- [ ] Standards correlation outline
- [ ] Project schedule

### 1.2 Manuscript Development

**Duration:** 16-24 weeks

1. Authors draft lesson content
2. Weekly check-ins with developmental editor
3. Rolling delivery by chapter
4. Integrate reviewer feedback

**Deliverables:**
- [ ] Draft manuscripts by chapter
- [ ] Problem sets and exercises
- [ ] Teacher notes and guidance

### 1.3 Content Review Cycles

**Duration:** 8-12 weeks (parallel with development)

| Review Type | Reviewer | Focus |
|-------------|----------|-------|
| SME Review | Math educators | Accuracy, pedagogy |
| Equity Review | DEI consultant | Representation |
| Accessibility | A11y specialist | WCAG compliance |
| Standards | Curriculum team | CCSS/state alignment |

## Phase 2: Editorial Production (Months 8-14)

### 2.1 Developmental Edit

**Duration:** 6-8 weeks

1. Structural editing for coherence
2. Verify mathematical accuracy
3. Ensure consistent terminology
4. Check cross-references

### 2.2 Copy Edit

**Duration:** 4-6 weeks

1. Grammar, spelling, punctuation
2. Style guide compliance
3. Citation verification
4. Fact checking

### 2.3 Design & Layout

**Duration:** 8-10 weeks (parallel)

1. Page design and templates
2. Art direction for illustrations
3. Photo research and licensing
4. Interactive design (digital)

## Phase 3: Production (Months 14-18)

### 3.1 Page Composition

**Duration:** 6-8 weeks

1. Typesetting in InDesign
2. Art placement and sizing
3. Table and chart formatting
4. Cross-reference linking

### 3.2 Proof Cycles

| Proof Stage | Focus | Duration |
|-------------|-------|----------|
| First Pages | Layout, art placement | 3 weeks |
| Revised Pages | Corrections applied | 2 weeks |
| Final Pages | Last check | 1 week |

### 3.3 Digital Production

**Duration:** 8-10 weeks (parallel)

1. ePub/PDF generation
2. Interactive element development
3. LMS integration setup
4. Accessibility testing

## Phase 4: Manufacturing & Delivery (Months 18-20)

### 4.1 Print Preparation

1. Pre-press file preparation
2. Color proofing
3. Print specification finalization
4. Printer selection and scheduling

### 4.2 Inventory & Distribution

1. Initial print run
2. Warehouse receipt
3. Fulfillment setup
4. Sample distribution

## Quality Gates

Each phase requires sign-off before proceeding:

| Gate | Approver | Criteria |
|------|----------|----------|
| G1: Manuscript Complete | Exec Editor | All content delivered |
| G2: Edit Complete | Managing Editor | All edits applied |
| G3: Pages Complete | Production Dir | Final pages approved |
| G4: Release | VP Editorial | Ready for market |
