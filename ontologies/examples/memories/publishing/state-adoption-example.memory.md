# Texas 2027 Mathematics Adoption - Into Math Submission

Strategic plan for submitting the Into Math K-8 program to the Texas
Proclamation 2027 mathematics adoption cycle.

## Adoption Overview

**State:** Texas
**Subject:** Mathematics (K-8)
**Adoption Type:** State-adopted (mandatory list)
**Contract Period:** 8 years (2027-2035)
**Market Size:** ~$180M over contract period

## Texas-Specific Requirements

1. **TEKS Alignment** (100% required)
   - All Texas Essential Knowledge and Skills covered
   - Vertical alignment documentation required

2. **Proclamation Requirements**
   - English and Spanish editions
   - Print and digital formats
   - Teacher editions and ancillaries
   - Assessment components

3. **Accessibility**
   - WCAG 2.1 AA compliance
   - Section 508 compliance
   - Screen reader compatibility

4. **Technology Requirements**
   - Single sign-on (SSO) support
   - Clever/ClassLink rostering
   - LTI 1.3 integration

## Submission Checklist

- [ ] TEKS correlation document (all grades)
- [ ] Accessibility VPAT for each component
- [ ] Technology specifications document
- [ ] Spanish language components
- [ ] Sample student editions (print)
- [ ] Sample teacher editions (print)
- [ ] Digital platform demo credentials
- [ ] Pricing schedule
- [ ] Publisher affidavit

## Review Panel Preparation

Based on previous cycles, focus preparation on:

1. **Mathematical rigor** - Show explicit connections to TEKS
2. **Coherence** - Demonstrate vertical progression
3. **Equity** - Diverse representation in content
4. **Teacher support** - Comprehensive PD materials

2. **Accessibility issues**: Complete VPAT review by Mar 1
3. **Spanish translation**: Finalize by Mar 15
4. **Technology demos**: Test platform stability by Apr 1
